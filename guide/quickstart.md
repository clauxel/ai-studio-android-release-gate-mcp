# Quickstart

AI Studio Android Release Gate is a hosted remote MCP for AI Studio Android release gate MCP.

## Fast Path

1. Open AI Studio Android Release Gate and select the buyer plan.
2. Create or request a bearer token from the hosted product.
3. Add https://aistudioandroidreleasegate.clauxel.com/mcp to a compatible MCP client.
4. Run tools/list, then call check_android_release_gate with public-safe sample data.
5. Save the returned receipt or export for human review.
